📦 Getting Started with GitHub Codespaces

GitHub Codespaces provides a complete development environment in your browser. No local setup required!

What is GitHub Codespaces?

GitHub Codespaces is a cloud-based development environment that runs directly in your browser. It gives you a full Visual Studio Code experience without installing anything on your computer.

1

Create a GitHub Account

If you don't have one already, go to github.com and sign up for a free account.

2

Create a New Repository

Click the + icon in the top right corner and select "New repository".

  • Give it a name (e.g., "my-first-website")
  • Add a description
  • Choose "Public" visibility
  • Check "Add a README file"
  • Click "Create repository"
3

Launch Codespaces

On your repository page:

  • Click the green <> Code button
  • Select the "Codespaces" tab
  • Click "Create codespace on main"
  • Wait for your environment to load (this may take 1-2 minutes)
4

Explore Your Codespace

You now have a full VS Code editor in your browser! The interface includes:

  • Explorer (left sidebar): Browse your files
  • Editor (center): Write your code
  • Terminal (bottom): Run commands
💡 Tip: Codespaces automatically saves your work. You get 60 hours of free usage per month!

🤖 Using GitHub Copilot

GitHub Copilot is your AI pair programmer that helps you write code faster and smarter.

What is GitHub Copilot?

GitHub Copilot is an AI-powered code completion tool that suggests entire lines or blocks of code as you type. It's like having an experienced developer helping you!

1

Enable GitHub Copilot

For students and educators, Copilot is free!

  • Go to github.com/settings/copilot
  • Click "Get access to GitHub Copilot"
  • If you're a student, apply for the GitHub Student Developer Pack at education.github.com
  • Once approved, Copilot will be automatically available in your Codespaces
2

Copilot in Action

Start typing in your editor and watch Copilot suggest completions in gray text.

  • Press Tab to accept a suggestion
  • Press Esc to dismiss a suggestion
  • Use Alt + ] or Option + ] to see the next suggestion
  • Use Alt + [ or Option + [ to see the previous suggestion
3

Write Code with Copilot

Try typing a comment describing what you want to do:

// Create a function that adds two numbers

Copilot will suggest the complete function! Accept it with Tab.

4

Best Practices

  • Write clear comments to get better suggestions
  • Give your variables and functions descriptive names
  • Review Copilot's suggestions before accepting them
  • Use Copilot to learn new patterns and techniques
💡 Tip: Copilot learns from your coding style. The more you use it, the better it gets at helping you!

🌐 Hosting with GitHub Pages

GitHub Pages lets you host your website directly from your GitHub repository - completely free!

What is GitHub Pages?

GitHub Pages is a free hosting service that turns your repository into a live website. Perfect for personal projects, portfolios, and documentation.

1

Prepare Your Website Files

Your repository needs at least an index.html file. This will be your homepage.

2

Commit Your Files

In your Codespace terminal, run these commands:

git add . git commit -m "Add website files" git push
3

Enable GitHub Pages

Go to your repository on GitHub:

  • Click on "Settings" tab
  • Scroll down and click "Pages" in the left sidebar
  • Under "Source", select "main" branch
  • Click "Save"
4

Access Your Website

After a few minutes, your site will be live at:

https://your-username.github.io/repository-name

The URL will be shown at the top of the Pages settings.

💡 Tip: Every time you push changes to your repository, GitHub Pages automatically updates your website!

🎯 Complete Tutorial: Build & Deploy Your First Website

Follow this complete step-by-step guide to create and publish your website!

Step 1: Set Up Your Project

1

Create Repository

Go to GitHub and create a new repository named "my-portfolio".

2

Open in Codespaces

Click Code → Codespaces → Create codespace on main.

Step 2: Create Your Website

3

Create index.html

In the Explorer, right-click and select "New File". Name it index.html.

4

Use Copilot to Write HTML

Type this comment at the top of your file:

<!-- Create a personal portfolio website with header, about section, and footer -->

Let Copilot suggest the HTML structure. Press Tab to accept suggestions!

5

Add Your Content

Customize the suggested code with your own information:

  • Your name and photo
  • About yourself
  • Your skills and projects
  • Contact information

Step 3: Preview Your Website

6

Open with Live Server

Right-click on index.html and select "Open with Live Server" or simply open the file in your browser.

7

Make Improvements

Don't like something? Use Copilot to help you:

  • Type /* Add a gradient background */ in your CSS
  • Type // Add smooth scrolling in your JavaScript
  • Let Copilot suggest the code!

Step 4: Deploy to GitHub Pages

8

Save Your Work

In the terminal (press Ctrl + ` to open), run:

git add . git commit -m "Create my portfolio website" git push
9

Enable GitHub Pages

Go to Settings → Pages → Select "main" branch → Save.

10

View Your Live Website!

Wait 1-2 minutes, then visit your site at:

https://your-username.github.io/my-portfolio

🎉 Congratulations! You've built and deployed your first website!

⚠️ Remember: Any changes you push to your repository will automatically update your live website. Always test locally first!
